# CollagePro - Static Image Collage Generator

A professional-grade, client-side image collage tool that runs entirely in the browser using HTML5 Canvas.

**Features:**
*   Smart Layouts (BSP Algorithm)
*   Fixed "Star" Image Support
*   Batch Generation (20 variations)
*   Visual Aspect Ratio Selection
*   Client-side Zipping & Downloading
*   No Server Required

## How to Deploy on Vercel

This is a static site with no build process.

1.  **Push to GitHub:** Upload these files (`index.html`, `README.md`) to a GitHub repository.
2.  **Import to Vercel:** Go to Vercel Dashboard -> Add New -> Project -> Import your repository.
3.  **Configure:**
    *   **Framework Preset:** Select **"Other"** (or just leave it default).
    *   **Build Command:** Leave empty.
    *   **Output Directory:** Leave empty (or `.`).
4.  **Deploy:** Click Deploy.

The site will be live instantly.

## Local Development

Simply open `index.html` in your web browser. No `npm install` or local server required.
